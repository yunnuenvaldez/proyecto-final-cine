# Cinema Management System

A web-based cinema management system developed with PHP and MySQL.

## Features

- User authentication
- Role-based access (Administrator and Client)
- Movie management
- Screening room management
- Show scheduling
- Ticket purchasing
- Seat selection
- Sales reports
- Database included

## Technologies

- PHP
- MySQL
- HTML
- CSS
- XAMPP

## Project Structure

